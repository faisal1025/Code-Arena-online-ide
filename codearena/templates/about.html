Welcome to

Code Arena is a coding platform to develop your curiosity in coding. It is a online platform which enables you to run your code online with out the need of Laptop or PC(Personal Computer). It has a Multi-Language text editor which supports languages like C, C++ and Python. It is built in very efficient manner that it will execute your code much faster and display the output as well as error if any bug found on the output area. It enables you to give a custom input in the input area if your code needs input to execute.

About Custom Input and Working

Custom Inputs are the inputs given by the user to run your the code if the code require user input. To give the user input first check the custom input checkbox the input section will appear on screen. Enter your input in the input box and click on run button. Bingo! Your code starts running the output will be shown in the output box. If there is any error in your code it will also shown in the output box.
